# TB - INC | Онлайн Библиотека

Полнофункциональное веб-приложение для онлайн-библиотеки на Next.js 16 с аутентификацией, хранением сессий и интеграцией с Open Library для обложек книг.

## ✨ Основные возможности

### 1. **Модальное окно входа/регистрации**
- ✅ Окно входа открывается автоматически если пользователь не авторизован
- ✅ Переключение между режимом входа и регистрации
- ✅ Валидация формы с сообщениями об ошибках
- ✅ Кнопка закрытия модального окна

### 2. **Аутентификация и управление сессией**
- ✅ Регистрация новых пользователей с хешированием пароля (bcryptjs)
- ✅ Вход с email и пароль
- ✅ Хранение сессий в HTTP-only cookies
- ✅ Сохранение состояния пользователя в localStorage (Zustand)
- ✅ Проверка сессии при загрузке страницы
- ✅ Кнопка выхода в заголовке

### 3. **Интеграция с Open Library**
- ✅ Загрузка реальных обложек по ISBN
- ✅ Поддержка всех форматов ISBN (10 и 13 символов)
- ✅ Fallback заглушки если обложка недоступна
- ✅ Сервис `book-service.ts` для работы с API Open Library

### 4. **Интерфейс библиотеки**
- ✅ 4 раздела с книгами (Романы, Детектив, Сказки, Фантастика)
- ✅ По 8 книг в каждом разделе (всего 32 книги)
- ✅ Горизонтальная прокрутка с кнопками листания
- ✅ Карточки книг с обложкой, названием, автором и рейтингом
- ✅ Поиск книг (подготовлен, можно расширить)
- ✅ Заголовок с названием сайта "TB - INC"

## 🗄️ Хранение данных

### Пользователи
- 📁 Файл: `data/users.json`
- 🔐 Пароли хешируются bcryptjs с salt=10
- 🍪 Сессия сохраняется в HTTP-only cookies (7 дней)

### Структура пользователя
```json
{
  "id": "random-id",
  "name": "Имя пользователя",
  "email": "user@example.com",
  "password": "bcrypt-hash"
}
```

## 🔌 API маршруты

### POST `/api/auth/register`
Регистрация нового пользователя
```json
{
  "name": "Имя",
  "email": "user@example.com",
  "password": "securepassword"
}
```

### POST `/api/auth/login`
Вход пользователя
```json
{
  "email": "user@example.com",
  "password": "securepassword"
}
```

### GET `/api/auth/session`
Проверка текущей сессии

### POST `/api/auth/session`
Выход из аккаунта (удаляет cookie)

## 📚 Как заменить ISBN обложек

Все ISBN хранятся в файле `components/library.tsx` в объекте `BOOKS_DATA`:

```typescript
const BOOKS_DATA: Record<string, Book[]> = {
  'Романы': [
    { 
      title: 'Война и мир', 
      author: 'Лев Толстой', 
      isbn: '0199232768',  // ← Замените здесь
      rating: 4.8, 
      genre: 'Романы' 
    },
    // ...
  ]
}
```

### Найти правильный ISBN:
1. Откройте [Open Library](https://openlibrary.org/)
2. Найдите книгу в поиске
3. Скопируйте ISBN со страницы книги
4. Замените значение в коде

### Формат:
```
https://covers.openlibrary.org/b/isbn/{ISBN}-M.jpg
```

Подробнее см. файл `public/ISBN_REFERENCE.md`

## 🎨 Компоненты

### `components/library.tsx`
Главный компонент библиотеки с сечениями книг и интерфейсом

### `components/book-card.tsx`
Карточка одной книги с обложкой, названием и рейтингом

Модальное окно для входа и регистрации

Сервис для работы с Open Library API и загрузки обложек

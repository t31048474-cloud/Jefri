# ISBN Справка для обновления обложек книг

## Как заменить ISBN в коде

Все книги хранятся в файле `components/library.tsx` в объекте `BOOKS_DATA`. Каждая книга имеет поле `isbn`, которое используется для загрузки обложки из Open Library.

## Как найти ISBN книги

1. Откройте [Open Library](https://openlibrary.org/)
2. Найдите нужную книгу в поиске
3. На странице книги найдите ISBN (обычно в левой панели)
4. Скопируйте ISBN без дефисов

Или используйте API Open Library:
```
https://openlibrary.org/api/books?bibkeys=ISBN:XXXXXXXXXXXX&format=json&jscmd=data
```

## Как обновить ISBN

1. Откройте файл `components/library.tsx`
2. Найдите секцию с книгой в `BOOKS_DATA`
3. Замените значение поля `isbn` на новый ISBN
4. Сохраните файл
5. Обложка автоматически обновится при загрузке страницы

## API Open Library для обложек

Обложка загружается по URL:
```
https://covers.openlibrary.org/b/isbn/XXXXXXXXXXXX-M.jpg
```

Размеры обложек:
- `-S.jpg` - маленькая (120px)
- `-M.jpg` - средняя (300px) - используется в приложении
- `-L.jpg` - большая (400px)

## Примеры корректных ISBN

- 10-символьные: 0486278182, 0809500124
- 13-символьные: 9785171056063, 0062693735

Все форматы поддерживаются Open Library.

## Выполненные требования

### 1. ✅ Модальное окно входа вместо страницы
- **Компонент:** `components/auth-modal.tsx`
- **Функции:**
  - Автоматически открывается для неавторизованных пользователей
  - Переключение между входом и регистрацией
  - Красивый модальный дизайн с полупрозрачным фоном
  - Русский интерфейс
  - Валидация формы

### 2. ✅ Хранение сессий в файле/БД
- **Решение:** HTTP-only cookies + файловое хранилище
- **Компоненты:**
  - `app/api/auth/session/route.ts` - управление сессией
  - `/data/users.json` - хранилище пользователей
  - `lib/auth-store.ts` - управление состоянием на клиенте
- **Безопасность:**
  - HTTP-only cookies защищают от XSS атак
  - Bcryptjs для хеширования паролей (10 salt rounds)
  - Никогда не отправляем пароль на клиент
  - Secure флаг для production окружения

### 3. ✅ Реальные обложки из Open Library
- **API интеграция:** `lib/book-service.ts`
- **Источник:** Open Library API (covers.openlibrary.org)
- **Как работает:**
  - Каждая книга имеет ISBN
  - Обложка загружается по URL: `https://covers.openlibrary.org/b/id/{ISBN}-M.jpg`
  - Fallback механизм если обложка недоступна
  - Реальные изображения для всех 32 книг

### Технологический стек
- **Frontend:** React 19, TypeScript, Tailwind CSS
- **Backend:** Next.js 16, Node.js
- **Состояние:** Zustand + localStorage + HTTP-only cookies
- **Безопасность:** bcryptjs для хеширования паролей
- **API интеграция:** Open Library API для обложек
- **Хранилище:** JSON файл (/data/users.json)

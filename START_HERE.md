## ❓ Частые вопросы

### Q: Как изменить обложку книги?
**A:** Обновите ISBN в файле `components/library.tsx`
- Найдите нужную книгу
- Замените значение `isbn`
- Обложка обновится автоматически
- Подробнее: [INTEGRATION_GUIDE.md](./INTEGRATION_GUIDE.md)

### Q: Как добавить новую книгу?
**A:** В файле `components/library.tsx` в `BOOKS_DATA`:
```typescript
{
  title: 'Название книги',
  author: 'Автор',
  isbn: 'ISBN номер',
  rating: 4.5,
  genre: 'Жанр'
}
```

### Q: Где хранятся пользователи?
**A:** В файле `data/users.json`
- Структура: массив объектов с id, name, email, password (хеш)
- Пароли хешированы - исходный пароль нельзя восстановить

### Q: Как добавить больше книг?
**A:** Добавьте в нужный раздел в `components/library.tsx`:
```typescript
'Новый жанр': [
  // Добавьте книги сюда
]
```

### Q: Как это развернуть на production?
**A:** Читайте [MIGRATION_TO_DATABASE.md](./MIGRATION_TO_DATABASE.md)
- Нужно заменить JSON на реальную БД (Neon/Supabase)
- Настроить environment variables
- Развернуть на Vercel

### Q: Что если обложка не загружается?
**A:** Проверьте:
1. ISBN правильный (откройте в браузере: `https://covers.openlibrary.org/b/isbn/ISBN-M.jpg`)
2. Open Library может быть недоступен временно
3. Используйте другой ISBN для этой книги

## 🚨 Что если что-то не работает?

### Модальное окно не открывается
```
1. Откройте DevTools (F12)
2. Смотрите консоль на ошибки
3. Проверьте что dev сервер запущен (pnpm dev)
```

### Обложки не загружаются
```
1. Проверьте ISBN в components/library.tsx
2. Откройте в браузере ISBN URL напрямую
3. Убедитесь что Open Library доступен
```

### Ошибка при регистрации
```
1. Проверьте что все поля заполнены
2. Email должен быть в формате xxx@yyy.com
3. Пароль минимум 6 символов
4. Email не должен быть уже зарегистрирован
```

### Сессия теряется при обновлении
```
1. Включите cookies в браузере
2. Убедитесь что http://localhost:3000 доступен
3. Очистите localStorage: localStorage.clear()
```

Больше помощи: [QUICK_START.md](./QUICK_START.md) → "Решение проблем"

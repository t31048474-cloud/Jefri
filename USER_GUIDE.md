# Руководство пользователя TB - INC

## Быстрый старт

1. **Откройте приложение**: http://localhost:3000
2. **Войдите или создайте аккаунт**:
   - Email: `ivan@test.com`
   - Пароль: `password123`

## Основные функции

### 1. Переключение темы

В верхнем правом углу библиотеки найдите кнопку переключения темы:
- **Жёлтое солнце** = Светлая тема
- **Серая луна** = Тёмная тема

Нажмите кнопку для переключения. Ваш выбор будет сохранён в браузере.

### 2. Поиск книг

Используйте поле поиска "Поиск книги..." в header:
- Вводите название книги
- Результаты обновляются в реальном времени
- Сортировка по жанрам (Романы, Детектив, Сказки, Фантастика)

### 3. Чтение книги

1. **Выберите книгу** - нажмите на карточку книги в библиотеке
2. **Откроется страница читалки** с информацией о книге
3. На странице книги вы увидите:
   - **Обложка** - реальная обложка из Open Library
   - **Название и автор**
   - **Количество страниц** (если доступно)
   - **ISBN** - уникальный идентификатор книги
   - **Описание** - полное описание содержания

4. **Кнопки действия**:
   - **"Читать онлайн"** - откроет полный текст в Google Books (требует интернета)
   - **"Печать"** - сохранит страницу в PDF
   - **Стрелка назад** - вернёт в библиотеку

### 4. Профиль пользователя

В верхнем правом углу отображается:
- Ваше имя
- Ваш email
- **Кнопка выхода** (иконка стрелки)

Нажмите кнопку выхода для выхода из аккаунта. Вам нужно будет снова войти.

## Особенности

### Реальные данные
- Все обложки книг загружаются из Open Library API
- Описания книг основаны на реальной информации
- Каждая книга имеет уникальный ISBN

### Автоматическое сохранение
- Тема сохраняется автоматически в localStorage
- Сессия хранится в cookies
- Ваше состояние сохраняется при закрытии браузера

### Работает на всех устройствах
- Оптимизировано для мобильных телефонов
- Красиво отображается на планшетах
- Полная функциональность на десктопе

## Часто задаваемые вопросы

### Q: Как я могу создать новый аккаунт?
A: На странице входа нажмите "Зарегистрироваться" и следуйте инструкциям.

### Q: Могу ли я читать книги полностью?
A: Да! Нажмите кнопку "Читать онлайн" на странице книги. Если книга доступна в Google Books, вы сможете читать её там.

### Q: Как я сохраню книгу в формате PDF?
A: Откройте страницу книги и нажмите "Печать". Ваш браузер позволит сохранить как PDF.

### Q: Почему я вижу пустые обложки?
A: Если обложка недоступна, она будет показана как пустой прямоугольник. Это означает, что данные о книге в Open Library могут быть неполными.

### Q: Как я могу изменить мою тему снова?
A: Нажмите кнопку переключения темы в верхнем правом углу. Ваш выбор будет сохранён.

### Q: Что делать если я забыл пароль?
A: В этом приложении нет функции восстановления пароля. Вам нужно создать новый аккаунт с другим email.

## Технические детали (для разработчиков)

### Архитектура
- **Frontend**: React 19 + Next.js 16 + Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: JSON file (можно заменить на SQL БД)
- **Auth**: Bcrypt + HTTP-only cookies

### API endpoints
- `GET /api/book/[isbn]` - получить информацию о книге
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход
- `GET /api/auth/session` - проверка сессии
- `POST /api/auth/session` - выход

### Интеграции
- **Open Library API** - обложки книг
- **Google Books API** - описания и полный текст

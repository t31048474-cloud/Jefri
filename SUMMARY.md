**Решение:**

- ✅ Автоматическое открытие если пользователь не авторизован

- ✅ Кнопка закрытия (X)
- ✅ Валидация формы
- ✅ Сообщения об ошибках на русском

**Как работает:**
```
app/page.tsx → useAuthStore.checkSession()
    ↓
Сессия найдена? ──YES──→ Показать Library
    ↓NO
Показать AuthModal
    ↓
Пользователь входит/регистрируется
    ↓
Закрыть модальное окно → Показать Library
```

### ✅ 2. Хранение сессий в файле/БД

**Требование:** Добавить хранение сессий в файле/БД

**Решение:**
- ✅ HTTP-only Cookies для безопасного хранения
- ✅ File-based хранилище (`data/users.json`) для простоты
- ✅ Zustand для клиентского состояния
- ✅ localStorage для восстановления сессии
- ✅ Bcryptjs для хеширования паролей

**Механизм:**
```
1. Пользователь регистрируется/входит
2. Сервер хеширует пароль (bcryptjs)
3. Сервер сохраняет пользователя в data/users.json
4. Сервер устанавливает HTTP-only cookie (7 дней)
5. Клиент сохраняет пользователя в Zustand + localStorage
6. При перезагрузке: checkSession() восстанавливает из cookie
```

**API маршруты:**
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход
- `GET /api/auth/session` - проверка сессии
- `POST /api/auth/session` - выход

---

### ✅ 3. Реальные обложки из Open Library

**Требование:** Подключить реальных обложек из Open Library (заменить cover URL'ы)

**Решение:**
- ✅ Интеграция с Open Library API
- ✅ Загрузка обложек по ISBN
- ✅ Сервис `book-service.ts` для работы с API
- ✅ URL формат: `https://covers.openlibrary.org/b/isbn/{ISBN}-M.jpg`
- ✅ Fallback механизм если обложка недоступна

**Как использовать:**
```typescript
// Каждая книга имеет ISBN
const book = {
  title: 'Война и мир',
  author: 'Лев Толстой',
  isbn: '0199232768',  // ← Используется для обложки
  rating: 4.8,
  genre: 'Романы'
}

// Обложка загружается автоматически:
// https://covers.openlibrary.org/b/isbn/0199232768-M.jpg
```

## 🧪 Тестирование

### В браузере
1. Откройте http://localhost:3000
2. Модальное окно откроется автоматически
3. Зарегистрируйтесь или войдите
4. Просмотрите библиотеку

### Через API
```bash
# Регистрация
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Test","email":"test@example.com","password":"Password123"}'

# Вход
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"Password123"}'

# Проверка сессии
curl -X GET http://localhost:3000/api/auth/session
```

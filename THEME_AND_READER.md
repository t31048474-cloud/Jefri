# Тёмная/Светлая тема и читалка книг

## Завершённые требования

### 1. ✅ Тёмная и светлая тема
- **ThemeProvider** - Context-based система управления темой
- **ThemeToggle** компонент - кнопка переключения темы в header
- **localStorage** - сохранение предпочтений пользователя
- **System preference detection** - автоматическое определение системной темы
- **Full dark mode support** - все компоненты поддерживают обе темы с классами `dark:*`

### 2. ✅ Функциональность чтения книг
- **Book Reader страница** - красивый интерфейс для просмотра информации о книге
- **Динамические маршруты** - `/book/[isbn]` для каждой книги
- **Google Books интеграция** - получение реального контента из Google Books API
- **Fallback описания** - хардкодированные описания для основных книг
- **Кнопка "Читать онлайн"** - ссылка на полный текст в Google Books
- **Информация о книге** - количество страниц, ISBN, описание

### 3. ✅ Реальные данные
- **Open Library API** - все обложки загружаются из Open Library
- **Уникальные ISBN** - каждая книга имеет уникальный идентификатор
- **Нет повторений** - каждая книга в библиотеке разная
- **Реальные описания** - информация взята из реальных источников

## API интеграция

### Google Books API
- Используется для получения описания книг
- Бесплатный API без ключа для ограниченного количества запросов
- Fallback на хардкодированные описания если API недоступен

### Open Library API
- Используется для загрузки обложек книг
- Полностью бесплатный и без ограничений
- CDN URL: `https://covers.openlibrary.org/b/isbn/{isbn}-L.jpg`
